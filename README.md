# React-comments
React commenting system

[![Netlify Status](https://api.netlify.com/api/v1/badges/d7f65515-38ea-4a18-98a0-87c217cd14b8/deploy-status)](https://app.netlify.com/sites/react-comments/deploys)

## Installation

1. Clone this repo
2. `yarn install`
3. `cd server`
4. `npm run dev`
